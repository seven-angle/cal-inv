# SmartCalc — Interest Calculator

🔗 **[Live App → https://seven-angle.github.io/cal-inv/](https://seven-angle.github.io/cal-inv/)**

A client-side interest calculator for high-precision financial tracking. Record credits and debits across a timeline and get accurate interest calculations using **Flat** or **Daily** mode — then export a clean statement as PDF.

---

## 📌 What Is This For?

Designed for individuals, accountants, and small businesses who need to calculate interest on a running balance with mid-period transactions (credits/debits). No server, no signup — everything runs in your browser.
